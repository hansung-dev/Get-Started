
# Pycaret

PyCaret은 파이썬의 오픈소스, 로우코드 머신러닝 라이브러리로, 가설에서 통찰력으로 사이클 타임을 줄이는 것을 목적으로 한다.

PyCaret is an open source, low-code machine learning library in Python that aims to reduce cycle time from hypothesis to insights.
